# Simplify YouTube

Simplify YouTube is a Chrome extension that allows you to customize your YouTube experience by hiding various user interface elements such as player controls, header buttons, and sidebar sections. By removing distractions, you can focus purely on the content you want to watch.

## Features

This extension provides a popup menu where you can toggle the visibility of the following YouTube elements:

### Video Player Controls
*   Play/Pause
*   Next / Previous
*   Volume
*   Timecode
*   Chapters
*   Autoplay (Autonav)
*   Subtitles
*   Settings
*   Picture-in-Picture (PiP)
*   Theater mode (Size)
*   Play on TV (Remote)
*   Full screen
*   Tooltips

### Top Header
*   Voice Search
*   Create Button
*   Notifications
*   Account Profile

### Sidebar
*   Home & Shorts
*   Subscriptions
*   Library (History, Playlists, Watch Later, Liked Videos, etc.)
*   Explore Section
*   More from YouTube (Premium, Music, Kids)
*   Report History
*   About...

## Usage
1. Pin the extension to your Chrome toolbar for easy access.
2. Navigate to YouTube (`https://www.youtube.com/`).
3. Click the extension icon to open the settings popup.
4. Use the toggles to hide or show specific elements. Changes are applied automatically.

## Supported Languages
*   English (en)
*   Japanese (ja)
